# Image Gallery

## Project Description
This is a simple and responsive Image Gallery developed using HTML, CSS, and JavaScript. The project displays images in a grid layout and allows users to view them in a lightbox with Next and Previous navigation. It also includes category filters, hover effects, and responsive design.

## Features
- Responsive image gallery
- Lightbox image preview
- Next and Previous navigation
- Category filtering (All, Nature, Animals, City)
- Hover zoom effect
- Smooth transitions
- Mobile-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the project.
2. Place all images inside the `images` folder.
3. Open `index.html` in any web browser.
4. Or use VS Code with the Live Server extension.
